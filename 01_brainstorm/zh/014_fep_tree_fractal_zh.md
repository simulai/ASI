# FEP树：分形自组织作为AGI宏观FEP层

> 来源：FARS理论讨论——FEP宏观层是什么
> 日期：2026-03-24

## 核心假设

**每个元胞/模块都是完整的FEP系统**，高层FEP是低层FEP的统计汇聚。宏观FEP不是中央控制器，而是整棵FEP树的整体能量景观。

类比：树不是设计出来的，是能量梯度驱动生长的。

```
        树叶（任务FEP：具体推理）
           /|\
          / | \       ← 分枝（领域FEP：检索/生成/规划）
         /  |  \
        /   |   \       枝（模块FEP：Hopfield / CAO / HiPPO）
       /    |    \
      /     |     \
    ════════════════════  树干（全局FEP：能量景观主谷）
    ════════════════════  树根（基质FEP：Landauer / 互补配对）
```

## 每层的热力学含义

| 层次 | 类比 | FEP操作 | 时间尺度 |
|------|------|---------|---------|
| 树根 | Landauer耗散 | 物理层自由能最小化 | ~10⁻¹³s |
| 树干 | Hopfield吸引子 | 全局面能量谷形成 | ~收敛时间 |
| 树枝 | 模块化FEP | 局部子能量谷分裂 | ~训练迭代 |
| 树叶 | 任务推理 | 浅层自由能寻谷 | ~推理时间 |

## 支持证据

- 每个细胞是自包含的代谢系统（不需要中央控制器）
- Helix里每个碱基对都在执行互补配对（局部Landauer门）
- M1.4：Gs硬度梯度 = 完美的单调专业化（多头去相关）

## 待验证假设

### H-Fractal-1：局部vs全局FEP吸引子同构
- 测量：提取单个元胞激活模式 → 计算吸引子维度
- 对比：全系统吸引子维度
- 预期：局部吸引子维度 < 全局，但结构同构

### H-Fractal-2：临界耦合强度
- 测量：调节层间连接权重 → 测量局部/全局最优占比
- 预期：存在临界耦合，平衡局部自主性与全局一致性

### H-Fractal-3：Helix分形验证（最直接）
- 操作：多层Helix（3-5层）→ 测量各层吸引子结构
- 预期：层数增加，吸引子维度增加但类型不变

### H-Tree-1：拓扑可塑性
- 设置：固定总参数，允许网络"生长"结构
- 测量：各层参数量分布
- 预期：接近幂律分布 → 支持树假设

### H-Tree-2：自愈能力
- 操作：剪枝 → 重新训练
- 预期：结构自发恢复 → 支持自组织假设
