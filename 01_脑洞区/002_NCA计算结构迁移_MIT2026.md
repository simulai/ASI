# MIT NCA Pre-pre-training 的热力学解释

> 来源：MIT Improbable AI Lab, Pulkit Agrawal团队
> 论文：arXiv:2603.10055 (2026-03)
> 日期：2026-03-23

## 实验结果

- NCA数据（1.64亿token）> 16亿英语token的效果
- 语言模型困惑度降低最多6%
- 收敛速度加快最多1.6倍
- 注意力层迁移效果最好（重新初始化注意力=最大性能损失）
- 复杂度匹配：OpenWebText/OpenWebMath偏好高复杂度NCA，代码偏好中等复杂度

## MIT的解释

"语言模型真正需要学习的，可能不是语言本身，而是语言背后的计算结构"

## 我们的热力学解释

**NCA轨迹 = 热力学纯粹的吸引子演化**

- CA的每个像素不是"语义token"，而是"吸引子状态"
- NCA每条序列来自随机规则 = 每个吸引子basin不同
- 预测下一个像素 = 推断当前吸引子动力学 = 在自由能landscape上做梯度下降
- 自然语言的"语义捷径"和"共现先验" = 在自由能landscape上有局部最小值可依赖 = 不需要做纯粹推断

**Landauer视角**：每一步CA更新 = 擦除1bit信息 = 必须耗散kT·ln2
自然语言的token有统计冗余 = 信息未充分擦除 = 训练效率低
NCA数据每个token都强制充分擦除 = 训练效率高

## 关键推论

1. **Attention层的迁移性最好** = Attention ≈ 自由能最小化的通用实现 = 不依赖领域
2. **MLP层迁移效果差** = MLP = 存储领域特定模式 = 依赖源域和目标域匹配
3. **复杂度匹配决定迁移效果** = 自由能landscape的"形状"（曲率）决定哪种吸引子对目标最有价值
